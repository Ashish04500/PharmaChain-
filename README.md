# PharmaChain

PharmaChain is a blockchain-based drug authenticity tracking system designed to prevent counterfeit medicines in the pharmaceutical supply chain.

## Features

- Drug batch registration on blockchain
- Ownership transfer between stakeholders
- QR code generation for each batch
- Consumer verification through QR scanning
- Transparent supply chain tracking

## Tech Stack

Frontend:
- React (Vite)
- CSS
- Ethers.js

Blockchain:
- Solidity
- Ethereum Sepolia Testnet

Tools:
- MetaMask
- Remix IDE
- QRCode.react
- html5-qrcode

## Setup Instructions

### Clone the Repository


git clone <repo-url>


### Install Dependencies


cd pharmachain-frontend
npm install


### Run the Project


npm run dev


## Stakeholders

- Manufacturer
- Distributor
- Pharmacy
- Consumer

## Demo Flow

1. Manufacturer creates drug batch
2. QR code generated
3. Distributor transfers ownership
4. Pharmacy verifies drug
5. Consumer scans QR to verify authenticity
